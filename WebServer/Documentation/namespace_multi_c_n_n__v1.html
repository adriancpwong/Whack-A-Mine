Detailed Description

Creates the multi class neural network model from scratch and fits to the data input, only uses 3 classes.

Used for the training of the multi class neural network. Creates a sequential neural network, and adds it's respective layers. The model is then compiled, fit to the training data, and then saved to be used by the prediction scripts. Does not use pre-existing model.
