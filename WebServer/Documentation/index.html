Image Recognition software developed with Keras and TensorFlow to distinguish between categories supplied in the Kaggle dataset.

Requirements

  • Python 3.5
  • Keras 2.1.1
  • Tensorflow 1.4.0
  • Reduced data set

How to run:

  • Ensure CNN.py train_data_dir & validation_data_dir point to the correct folders from the reduced data set.
  • python CNN.py
